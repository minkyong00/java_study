# [학점관리모델링]
# 간단한 학점관리 시스템의 모델링을 통해 학생의 과목별 성적을 관리하고
# 학점계산정책에 따라 학점을 출력하는 프로그램을 작성

# 주요 클래스 및 역할
1. User (추상클래스) : id(아이디), name(이름), printInfo(정보출력메소드)
2. Student (클래스) : 학생의 정보 및 수강과목 정보를 관리하는 클래스
3. Subject (클래스) : 과목, 과목마다 학점 정책을 설정할 수 있음
4. Enrollment (클래스) : 수강내역, 학생이 수강한 과목과 점수를 저장
5. GradePolicy (인터페이스) : 학점 계산 정책 인터페이스
6. BasicGradePolicy (클래스) : GradePolicy를 구현한 클래스
    90점 이상이면 A, 80점 이상이면 B, 70점 이상이면 C, 60점 이상이면 D, 나머지 F
7. PFGradePolicy (클래스) :  GradePolicy를 구현한 클래스
    70점 이상이면 "PASS" 그렇지 않으면 "FAIL"
    
 # 기능 요구사항
 1. 학생 등록 : 아이디와 이름을 입력하여 학생 객체 생성
 2. 과목 등록 : 과목명과 학점 정책을 입력하여 과목 객체 생성
 3. 수강 등록 : 특정 학생에게 과목과 점수를 등록
 4. 성적 출력 : 각 학생의 수강과목별 학점을 출력
 
 # 출력 예
 홍길동 님의 성적
 과목: 수학 / 점수: 85 / 학점: B
 과목: 윤리 / 점수: 65 / 학점: FAIL
